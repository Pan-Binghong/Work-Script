以下是Jupyter Notebook的一些常用快捷操作：

命令模式

在Jupyter Notebook中按下 Esc 键即进入命令模式，此时单元格外侧边框变为蓝色。

A：在当前单元格上方添加一个新单元格。
B：在当前单元格下方添加一个新单元格。
DD：删除当前单元格。
M：将当前单元格切换为Markdown模式。
Y：将当前单元格切换为代码模式。
1 ~ 6：将当前 Markdown 单元格变为对应的标题（# ~ ######）。
H：显示所有快捷键。
编辑模式

在Jupyter Notebook中按下Enter键，即进入编辑模式，此时单元格外侧边框变为绿色。

Ctrl + Enter：运行当前单元格，并选中下面的单元格。若是最后一个单元格则不选中。
Shift + Enter：运行当前单元格，并选中下面的单元格。若是最后一个单元格，则新建一个单元格并选中该单元格。
Alt + Enter：运行当前单元格，并在下面插入一个新单元格，并选中该单元格。
Ctrl + /：注释或取消注释当前行或选中区域。
Shift + Tab：显示函数或类的参数和文档字符串。重复按下时，分别显示更多信息。
Ctrl + Shift + -：在光标所在行下方分割单元格。
Ctrl + Shift + +：在光标所在行上方分割单元格。
以上仅是一些常用的快捷键，你也可以在Jupyter Notebook的 Help -> Keyboard Shortcuts 中看到完整的快捷键列表，你也可以自定义快捷键的方式来使得自己更舒适使用。
